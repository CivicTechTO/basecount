# Basecount
Helping respite centres communicate their population counts. This is a **citizen-led** project.

## Starting the app

## Technologies Used
- [**Ruby on Rails**](https://rubyonrails.org/)
- [**Node JS**](https://nodejs.org/)
- [**React**](https://reactjs.org/)

### Requirements
- ruby 2.4.1
- node JS

### Local Development
```
  bundle install;
  npm install;
  rails s;
```
